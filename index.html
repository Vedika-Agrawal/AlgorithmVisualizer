Sorting Algorithms

A Sorting Algorithm is used to rearrange a given array or list of elements according to a comparison operator either in ascending order or descending order. Lets Visualize how sorting work.

Tree Traversals

Tree traversal (also known as tree search and walking the tree) is a form of graph traversal and refers to the process of visiting (e.g. retrieving, updating, or deleting) each node in a tree data structure, exactly once.

Graph Algorithms

Graph algorithms are a set of instructions that traverse (visits nodes of a) graph. Some algorithms are used to find a specific node or the path between two given nodes. Lets Visualize and understand.
